DBBrowser

This application allows you to browse the contents of relational databases using a web browser. No knowledge of SQL is required, although you can use the DBBrowser for issuing SQL queries.

Help on Servers Page

This page lists the database servers that you can browse. From this page you can:
  1. Select a server to browse.
  2. Add a server to the list of those available.
  3. Delete a server from the list of those available.
  4. Shutdown the DBBrowser application.
